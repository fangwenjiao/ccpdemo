====================Install目录安装包使用方法====================

1.登录www.cloopen.com获取到Demo账号信息，将账号信息填写到工程res\raw\config.properties；

2.拷贝config.properties文件到手机SD卡的根目录（注意：如果手机有2个SD卡，请使用手机本身SD卡）

3.如果填写多个账号，各个账号之间以英文逗号分隔
